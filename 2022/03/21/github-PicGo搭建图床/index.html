github+PicGo搭建图床

2022-03-21 其他 298 字
github

我们在写博客的时候往往都要插入很多的图片，本地的图片一般都都会失败，因为平台不会自动给你上传，这时我们就需要一个图床来帮我们存储图片，我们上传到图床之后，就可以任意的使用，也不用担心图片丢失。

1.搭建过程

因为我们是要把图片上传到github上，所以我们要在github上单独建一个库专门来放我们的图片。

1.登入自己的github，创建新的库。

2.填写库的资料

3.创建好库之后，我们再去创建一个token

创建token路径 

Settings -> Developer settings -> Personal access tokens -> generate new token;

只需要勾选repo 其他默认

4.然后我们就可以得到token（复制保存好，后面要用，注意它只会显示一次）

2.下载PicGo

1.我们先去下载PicGo

paru -S picgo-appimage

2.配置PicGo，依次打开 图床设置 -> GitHub图床

3.填写相关信息，最后点击确定。

3.总结

我们搭建图床就搭建完成了，相信对我们以后写博客一定是事半功倍！！！
