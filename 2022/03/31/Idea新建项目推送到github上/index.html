Idea新建项目推送到github上

2022-03-31 其他 188 字
github

原先都是用终端来把项目推送到github上，但是每次输入密码都不行，都要输入token，感觉太麻烦了就学习了下用idea来推送上去

1.用idea打开要推送的项目

2.将本地项目创建一个git仓库

3.创建了git库后就可以发现所有的文件都成了待提交状态

（在这里需要登入一下github，因为我登入过我的github，如果没有登入过的话要先登入一下）

4.点击Add等待一会就可以发现提交成功了

（可以去github上刷新查看了）

总结

用idea部署项目实在太方便了，甚至不用去github上自己手动建库；
